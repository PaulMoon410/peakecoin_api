PeakeCoin API JS & JSON Directory

Browse and use all PeakeCoin API modules below. Click to view or copy CDN links. For usage, see examples below each entry.

  • announcements
    JS Module | JSON
    Site-wide announcements and news.
    announcements.announcements
  • audit_log
    JS Module | JSON
    Audit log entries for admin or compliance review.
    auditLog.logs
  • blockLog
    JS Module | JSON
    Static/fallback log of blockchain blocks processed by the PeakeCoin system.
    blockLog.blocks
  • keychain
    JS Module | JSON
    Helpers for Hive Keychain signature requests.
    keychainHelpers.requestSignBuffer(...)
  • hive_signer
    JS Module | JSON
    Helpers for generating HiveSigner.com URLs for login/auth and transaction signing.
    hiveSigner.getLoginUrl(...)

How to Use

For browser/JS:
<script src="https://cdn.jsdelivr.net/gh/PaulMoon410/peakecoin_api/api_listing.js"></script>
console.log(window.apiModules);

For Python or other languages:
import requests
modules = requests.get('https://cdn.jsdelivr.net/gh/PaulMoon410/peakecoin_api/api_listing.json').json()

All modules are public and can be used for read-only access from any static site or backend.
